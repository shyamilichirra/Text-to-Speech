# Text-to-Speech

A simple text-to-speech application that converts written text into spoken language.

## Features
- Converts text to speech in real-time
- Supports multiple languages and voices
- Easy-to-use interface

## Usage
1. Clone the repository.
2. Open `http://127.0.0.1:5500/index.html` in your browser.
3. Enter text and click "Speak".

## Example
- Input: "Hello, how are you?"
- Output: Speech playback of the text
